TM WEEKLY PACK/IDO Processing Instructions

📦 PACK/IDO Processing Workflow

🚀 Processing Steps

  1. Drop original data files into DATA FILE DROP BOX
  2. Click the RUN INITIAL button
  3. Select a file for processing by clicking on one of the file names in the SELECT FILE FOR PROCESSING box
  4. Click the PROCESS INDIVIDUAL 1 button
  5. Open Bulk Mailer by clicking the BULK MAILER button
    1. TaskMaster GOJI TM WEEKLY & IDO 1
      1. Run
      2. Go to Views
        1. Select BASIC + USER TEXT
        2. Mass Delete all records
    2. TaskMaster GOJI TM WEEKLY & IDO 2
      1. Run
        1. If Current Count has more than 0 records go to Views
          1. Select BASIC + USER TEXT
          2. Fix addresses, CASS with F9
        2. Go to Groups (skip to c. if Current Count is 0)
          1. TRACHMAR DP NCOA & BAD CASS
        3. Go to Views
          1. BASIC + USER TEXT
          2. Change contents of User Text 3 to 14
    3. TaskMaster GOJI TM WEEKLY & IDO 3
      1. Run
  6. Click the PROCESS INDIVIDUAL 2 button
  7. Repeat steps 3 – 5 for all data files until they are all processed
  8. Click the RUN DPZIP button
    1. This will open a Windows Explorer window with access to the ZIP files
  9. Reply All to the email and attach ZIP files from the open folder
  10. Click the RUN DPZIP BACKUP button
  11. Close & delete TM LIST PROCESSING files in Bulk Mailer
✅ The process is now complete
