FOUR HANDS Processing Instructions

📋 Job Data Locked - Processing Ready

📝 Edit Note: If you need to edit the job data as a result of erroneous data entry, click the JOB DATA EDIT button, correct all incorrect data, then click the DATA LOCK button.

🚀 Processing Workflow

  1. Run the RUN INITIAL button
  2. Open Bulk Mailer
    1. TaskMaster GOJI FOUR HANDS 01
      1. Edit Create Mailing, change File
        1. Update job and drop numbers XXXXX FOUR HANDS XD
      2. Save, then Run
        1. In the event the list is short of 200 pieces go to Import
          1. Import APM PROP UP SEEDS
            1. These are in sets of 10; repeat until total record count is above 200
      3. Go to Address Quality on the left hand side
        1. Go to Data Services
          1. Click Next three times
          2. Check the box for Walk Sequence
          3. Click Next four times
          4. Click Finish
            1. Walk Sequence will run
            2. Once complete click Close
    2. TaskMaster GOJI FOUR HANDS 02
      1. Edit Mail Sort
        1. Mailing and Permit Information
          1. Mailing Date is today's date
          2. Update Statement Number if necessary
          3. Update Job/Reference ID drop number
        2. Sorting Preparation
          1. Documents
            1. Change Mailing Title XXXXX FOUR HANDS DX
            2. Update Carrier Route Sequencing date to today's date
        3. Tray Label Layout
          1. AMERCIAN PRINTING FH XXXXX DX
      2. Save, then Run
        1. Paperwork will print after
          1. Mail Sort Summary: Page 1 only
          2. NOTE: if after adding APM PROP UP SEEDS the final count is over 200 remove the excess APM seeds, then re-sort
  3. Update count and postage data
    1. Enter postage and piece count in Postage and Count boxes respectively
    2. Click POSTAGE LOCK button
      1. If you need to correct any postage information due to erroneous input simply click the POSTAGE LOCK button to unlock the data, correct the mistakes, then click the POSTAGE LOCK button again
  4. Click the FINAL STEP button
    1. Reply All to e-mail
      1. Copy and paste the top line from the FOUR HANDS LOG into the e-mail
    2. Submit same day
⚠️ Important: Follow each step carefully and ensure all data is correct before proceeding to the next step.
