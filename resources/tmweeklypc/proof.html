TM WEEKLY PC Proof Instructions

📋 Job Data Locked - Proof Processing

📝 Edit Note: If you need to edit the job data as a result of erroneous data entry, click the JOB DATA EDIT button, correct all incorrect data, then click the DATA LOCK button.

🚀 Proof Processing Workflow

  1. Click the RUN INITIAL button
  2. Open Bulk Mailer
    1. TaskMaster GOJI TM WEEKLY PC 1
      • Edit Create Mailing, change File
        1. Update job and week numbers as needed
      • Save, then click Run
      • Go to Views
        1. Select BASIC + USER TEXT
        2. Mass Delete all records
    2. TaskMaster GOJI TM WEEKLY PC 2
      • Run
      • Go to Views
        1. Select BASIC + USER TEXT
        2. Fix addresses, CASS with F9
      • Go to Groups (skip to c. if Current Count is 0)
        1. TRACHMAR DP NCOA & BAD CASS
      • Go to Views
        1. BASIC + USER TEXT
        2. Change contents of User Text 3 to 14
    3. TaskMaster GOJI TM WEEKLY PC 3
      • Edit Mail Sort
        1. Mailing and Permit Information
          • Change Mailing Date to given mail date
          • Change Statement Number once per month
          • Change Job Reference ID XX.XX
        2. Sorting Preparation
          • Documents
            • Change Mailing Title XXXXX XX.X FHK NEW MEMBERS
        3. Tray Label Layout
          • Company Name XXXXX XX.XX
      • Save, then Run
        • Duplicates
          • Prioritize Head of Household records
      • If there aren't enough pieces for the sort: accept, and continue; list will export, skip to 3)
      • Paperwork will print after
        • Mail Sort Summary: Page 1 only
        • Tray Labels: Preferences>Paper/Quality
          • Paper Source: Manual Feed (Tray 1)
          • Media Tray: Card Stock
  3. Update count and postage data
    1. Enter postage and piece count in Postage and Count boxes respectively
    2. Select appropriate postage class and permit from Class and Permit dropdown boxes
    3. Click POSTAGE LOCK button
      • If you need to correct any postage information due to erroneous input simply click the POSTAGE LOCK button to unlock the data, correct the mistakes, then click the POSTAGE LOCK button again
  4. Click the RUN PROOF DATA button
  5. Select Sorted or Unsorted from the dropdown menu underneath the OPEN PROOF FILE button, then click OPEN PROOF FILE
    • Proof file will open in Adobe InDesign
      • SmartStream Designer
        • Database
          • Click Reload
        • Create Job
          • Ensure Apply Imposition is unchecked
          • Create
  6. Click the RUN WEEKLY MERGED button
    • GOJI will produce a pop-up window displaying the PDF and CSV files in the following step
  7. Reply All to e-mail
    • attach FHK Weekly Merged CSV and PDF proof to e-mail
    • Right-click top row from WEEKLY PC LOG, click Copy, and paste into e-mail
⏳ Wait for proof approval
