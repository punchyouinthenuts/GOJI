(if you need to edit the job data as a result of erroneous data entry, click the JOB DATA EDIT button, correct all incorrect data, then click the DATA LOCK button)

  1. Click the RUN INITIAL button
  2. Open Bulk Mailer
    1. TaskMaster GOJI TM WEEKLY PC 1
      • Edit Create Mailing, change File
        1. Update job and week numbers as needed
      • Save, then click Run
      • Go to Views
        1. Select BASIC + USER TEXT
        2. Mass Delete all records
    2. TaskMaster GOJI TM WEEKLY PC 2
      • Run
      • Go to Views
        1. Select BASIC + USER TEXT
        2. Fix addresses, CASS with F9
      • Go to Groups (skip to c. if Current Count is 0)
        1. TRACHMAR DP NCOA & BAD CASS
      • Go to Views
        1. BASIC + USER TEXT
        2. Change contents of User Text to 14
    3. TaskMaster GOJI TM WEEKLY PC 3
      • Edit Mail Sort
        1. Mailing and Permit Information
          • Change Mailing Date to given mail date
          • Change Statement Number once per month
          • Change Job Reference ID XX.XX
        2. Sorting Preparation
          • Documents
            • Change Mailing Title XXXXX XX.X FHK NEW MEMBERS
        3. Tray Label Layout
          • Company Name XXXXX XX
      • Save, then Run
        • Duplicates
          • Prioritize Head of Household
      • If there aren’t enough pieces for the sort: accept, and continue; list will export
      • Paperwork will print after completion
        • Mail Sort Summary: Page 1 only
        • Tray Labels: Preferences>Paper/Quality
          • Paper Source: Manual Feed (Tray 1)
          • Media Tray: Card Stock
  3. Update count and postage data
    1. Enter the number in Postage and Count boxes respectively
    2. Select appropriate options from Class and Permit dropdown menus
    3. Click POSTAGE LOCK button
      • If you need to correct any postage information due to an error simply click the POSTAGE LOCK button to unlock the data, correct the errors, then click the POSTAGE LOCK button again
  4. Click the RUN PROOF DATA button
  5. Select Sorted or Unsorted from the dropdown menu underneath the OPEN PROOF FILE button, then click OPEN PROOF FILE
    • Proof file will open in the PDF
  6. Click the RUN WEEKLY button
    • Files will open in Windows Explorer windows to access necessary files
  7. Reply All to the e-mail
    • Attach FWC Weekly Merged CSV to the e-mail
    • Right-click the top row from WEEKLY PC LOG, click Copy to copy, and paste into the email
