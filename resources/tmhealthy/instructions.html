🚀 Processing Workflow

  1. Run the RUN INITIAL button
  2. Open Bulk Mailer
    1. TaskMaster TM HEALTHY BEGINNINGS 01
      1. Edit Create Mailing, change File
        1. Update job number
      2. Save, then Run
      3. Go to Views
        1. Select BASIC + USER TEXT
        2. Mass Edit all records
          1. Change the contents of a field
          2. Select Field User Text 3
          3. New Value 14
          4. Finish
            1. Click Yes
    2. TaskMaster TM HEALTHY BEGINNINGS 02
      1. Mass Edit all records
        1. Change the contents of a field
        2. Select Field User Text 3
        3. New Value 14
        4. Finish
          1. Click Yes
    3. TaskMaster TM HEALTHY BEGINNINGS 03
      1. Edit Mail Sort
        1. Mailing and Permit Information
          1. Change Mailing Date to drop date
          2. Update Statement Number
        2. Sorting Preparation
          1. Documents
            1. Change Mailing Title XXXXX TM HEALTHY BEGIN
        3. Tray Label Layout
          1. AMERCIAN PRINTING XXXXX TM HB
      2. Save, then Run
        1. Paperwork will print after
          1. Mail Sort Summary: Page 1 only
          2. Tray Labels: Preferences>Paper/Quality
            1. Paper Source: Manual Feed (Tray 1)
            2. Media: Card Stock
  3. Update count and postage data
    1. Enter postage and piece count in Postage and Count boxes respectively
    2. Click POSTAGE LOCK button
      1. If you need to correct any postage information due to erroneous input simply click the POSTAGE LOCK button to unlock the data, correct the mistakes, then click the POSTAGE LOCK button again
  4. Click the FINAL STEP button
    1. Reply All to e-mail
      1. Copy and paste link to print data file into the e-mail
      2. Drag & drop the merged data file from the pop-up window into the e-mail
      3. Copy and paste the top line from the HEALTHY BEGINNINGS LOG into the e-mail
⚠️ Important: Follow each step carefully and ensure all data is correct before proceeding to the next step.
