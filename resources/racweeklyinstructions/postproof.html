GOJI will open the proof ZIP file folder in Windows Explorer, then present a window with two tables.

  • One compares counts from original input files to the final sorted counts
  • The other will have all of the data you need for counts and postage
    • Click the Copy button to copy the second table
  • Open the email for this job
    • Reply All to the message. Right-click in the email body and click Paste. This will paste the table into the email body.
  • Go to the Windows Explorer window with the ZIP files
    • Select all of the ZIP files, then drag those to the email and drop them there to attach them
  • Once all of the proof ZIP files have uploaded to the email click Send

You are now in the Proof Approval phase. Save the job, and close if necessary. You will receive an email with proof approval status for each job. For each job that is completely approved check its box under PROOF APPROVAL. If all are approved simply click All. If any are not approved and require changes, follow the steps below:

  • Select the proof files you need to open using OPEN PROOF FILES and the drop down menu below it. Close any files you do not need to make any changes to.
    • Make necessary corrections and save as normal, overwriting the original files (if they are needed in the future, they still exist in the original proof ZIP files)
  • Click on PROOF REGEN LOCK, and select only the proof files you made updates/corrections to
  • Click RUN POST PROOF
    • This will generate new ZIP files that contain only the proofs you regenerated. They will have a vX identifier appended to the end of their file names.
  • Click on PROOF REGEN LOCK to disengage it
  • Reply to the email and attach those updated ZIP files only, updating the PROOF APPROVAL section accordingly
