1. Click the OPEN INPUTZIP button
    1. Move the data ZIP file from the Weekly email to this folder
    2. Close the Windows Explorer window
  2. Click the RUN INITIAL button; this creates home folders for each job, and extracts the data files from the ZIP and processes them to place them all in their respective working folders
  3. Open Bulk Mailer for data processing, then navigate to the TaskMaster Basic tab, and run the following templates first double-clicking on the template name, then selecting the task outlined in the instructions below, and clicking the Edit button on the right. Move through the pages updating information as necessary, and when all updates are complete as per the instructions right-click on the template name and select Run "[template]"...

          1. Change Mailing Date to Friday before Job Date
          2. Change Statement Number once per month
      3. Sorting Preparation
        1. Documents
          1. Change Mailing Title XXXXX XX.X CBC2
        2. Tray Label Layout
          1. AMERICAN PRINTING CBC2 XX.XX
