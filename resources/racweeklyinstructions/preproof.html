Just below OPEN PROOF FILES is a drop down menu. Click on that drop down menu to select one of five jobs: CBC, EXC, INACTIVE, NCWO or PREPIF. Once selected click the OPEN PROOF FILES button, and the files for that job will open in Adobe InDesign. For each proof file you open do the following:

  • CBC ONLY: Update the Date field on the first page (green graphic) to include the date of the job
  • Click on the menu item SmartStream Designer
    • Select Database, click Reload (you should see the text in the preview box change), then OK
  • Click on the checkbox for Preview in the toolbox on the right
    • Ensure all data is populated appropriately
    • For files where data is merged on multiple pages you will have to check one page at a time. Check all entries for the first page, uncheck Preview, then scroll to the other page, re-check Preview, and check all entries for that page.
  • Click on the menu item SmartStream Designer
    • Select Create Job, ensure Apply Imposition is unchecked, then click Create
  • Once the PDFs are created save the proof file, then close it inside of InDesign (leave InDesign open until you have completed all proofs)

Once all proof files have been created click RUN POST PROOF; this will ZIP up the proof files and create ZIP files in a specified folder and provide you with windows you need for the reply email you need to send.
