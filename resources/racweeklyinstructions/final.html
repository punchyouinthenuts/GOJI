With all proofs approved the last step is to generate the print files. The process is virtually identical to proof file generation; there are some slight differences so pay attention. You will not generate print files for INACTIVE.

Just below OPEN PRINT FILES is a drop down menu. Click on that drop down menu to select one of four jobs: CBC, EXC, NCWO or PREPIF. Once selected click the OPEN PRINT FILES button, and the files for that job will open in Adobe InDesign. For each print file you open do the following:

  • CBC ONLY: Update the Date field on the first page (green graphic) to include the date of the job
  • Click on the menu item SmartStream Designer
    • Select Database, click Reload (you should see the text in the preview box change), then OK
  • Click on the checkbox for Preview in the toolbox on the right
    • Ensure all data is populated appropriately
    • For files where data is merged on multiple pages you will have to check one page at a time. Check all entries for the first page, uncheck Preview, then scroll to the other page, re-check Preview, and check all entries for that page.
  • Click on the menu item SmartStream Designer
    • Select Create Job, ensure Apply Imposition is checked
    • For each job you will need to split the records so that the total sheet count is no greater than 250. Use the following values for each job:
      • CBC: 1250
      • EXC: 1750
      • NCWO: 1750
        • PR & APPR will typically not need their records split
      • PREPIF: 1750
        • PR will typically not need its records split
    • Click Create
  • Once the PDFs are created save the print file, then close it inside of InDesign (leave InDesign open until you have completed all print files)

Once all print files have been created click RUN POST PRINT; this will copy the print files to their respective folders on the network drive, then compress them locally for archival purposes. Once that process completes a window will pop-up with COPY and CLOSE buttons. Click COPY, then Reply All to the proof approval e-mail (ensure Preflight is included), and paste the copied contents of the window into the body of the e-mail, then click Send.

This job is now complete. Save and close it.
