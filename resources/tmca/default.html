📦 CA EDR/BA Processing Workflow

🚀 Processing Steps

  1. Drop original data files into DATA FILE DROP BOX
  2. Enter the job number into Job Number
  3. Select the current Year and Month from the drop down menus
  4. Click the DATA LOCK button to create a new file
  5. Click the PROCESS button
  6. Copy the postage table and the network path from the pop-up window
  7. Drag the MERGED files from the pop-up window to the email
  8. Click the CLOSE button on the pop-up window
✅ The process is now complete
